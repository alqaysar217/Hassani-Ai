# **App Name**: Hassani AI

## Core Features:

- Intelligent AI Chat Interface: A full-screen, mobile-first chat UI supporting seamless interaction between user and AI, capable of displaying diverse content types like text, code blocks, images, audio players, and structured diagrams.
- Context-Aware AI Intent Detection: Automatically analyzes user input to infer the desired intent (e.g., image generation, coding assistance, music creation, diagramming, or general questioning) and directs the request to the appropriate generative AI tool.
- Integrated Generative AI Toolkit: Leverages multiple AI models to provide tools for generating comprehensive text answers, creating images, composing music, synthesizing code with explanations, and generating structured diagrams (e.g., Mermaid syntax) based on detected user intent.
- Dynamic Media Input & Display: Enables users to attach files or images to their prompts and intelligently renders rich media content within the chat, including interactive audio players for generated music, code blocks with copy functionality, and visual output for diagrams and images.
- Persistent Chat History Management: Provides a sidebar for accessing, organizing, renaming, and deleting past conversations, allowing users to effortlessly pick up where they left off or revisit previous interactions.
- Personalized User Settings: Offers a dedicated settings page for customizing the app experience, including toggling between dark and light themes, adjusting language preferences, and securely managing external AI API keys.

## Style Guidelines:

- Color scheme is dark-first to promote a calm, focused, and 'dark mode friendly' user experience. This design choice fosters mental clarity.
- Primary accent color (interactive elements, highlights): Purple, carefully chosen to convey a sophisticated and creative yet calming energy. (#AC4DF0)
- Background color: A very dark, almost black hue with a subtle purple undertone for depth and to harmonize with the primary color, ensuring minimal distraction. (#1E1B1F)
- Secondary accent color: A deep blue, providing a distinct yet analogous contrast to the primary purple, suitable for differentiating key actions or information. (#6B6BF0)
- Headline and Body text font: 'Inter' (sans-serif), chosen for its modern, clean, and highly readable characteristics, mirroring the neutral and objective aesthetic of leading AI platforms.
- Code font: 'Source Code Pro' (monospace sans-serif), selected for optimal legibility and formatting of code snippets, enhancing the programming assistance feature.
- Icons should be minimalist, outline-based, and consistently rounded to align with the soft, modern aesthetic. Focus on clarity and ease of recognition.
- Mobile-first full-screen chat interface with ample spacing and a '2xl' rounded corner radius across UI elements, ensuring a smooth, organic feel similar to Google Gemini's clean design.
- Incorporate subtle, smooth transitions and micro-animations for message loading, conversation switching, and interactive elements to enhance the user experience and convey responsiveness without being intrusive.
